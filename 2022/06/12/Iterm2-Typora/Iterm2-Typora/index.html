Item2

下载的软件总是安装不上（大概是因为更新了系统）

brew install iterm2 (用brew下载)

配色方案

https://github.com/mbadolato/iTerm2-Color-Schemes

对应的iterm2文件夹是‘Schemes’

双击即可安装；

然后在Item2的Preferences里设置配色

其他个性化

  • Color 中 Cursor guide（光标导向器）
  • Text中 Vertical bar（竖线）& Blinking cursor（闪动）
  • Window 中Transparency（透明度）

powerlevel 10k

https://github.com/romkatv/powerlevel10k

  • 字体

https://github.com/ryanoasis/nerd-fonts

法一：直接下载双击安装（新系统失败了～）

法二：终端下载 （新系统又失败了～）

git clone https://github.com/powerline/fonts.git --depth=1

cd fonts

./install.sh

cd ..

rm -rf fonts

法三：先不安装，在插件设置的时候，自动安装（开了VPN）

  • Installation

mkdir .zsh

git clone --depth=1 https://gitee.com/romkatv/powerlevel10k.git ~/.zsh/powerlevel10k
echo 'source ~/.zsh/powerlevel10k/powerlevel10k.zsh-theme' >>~/.zshrc

  • Set up

首次安装重启进入设置；其他可以输入p10k configure 10k

zsh-autosuggestions代码自动补全

https://github.com/zsh-users/zsh-autosuggestions

git clone https://github.com/zsh-users/zsh-autosuggestions ~/.zsh/zsh-autosuggestions

echo 'source ~/.zsh/zsh-autosuggestions/zsh-autosuggestions.zsh' >> ~/.zshrc

zsh-syntax-highlighting语法高亮

https://github.com/zsh-users/zsh-syntax-highlighting

git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ~/.zsh/zsh-syntax-hightlighting

echo "source ${( q-)PWD}/.zsh/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh" >> ${ZDOTDIR:-$HOME}/.zshrc
